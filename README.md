# Rss-To-Json

This project is a simple RSS to JSON converter. It is built using NestJS and uses the [rss-parser](https://www.npmjs.com/package/rss-parser) package to parse the RSS feed.

## Project setup

```bash
yarn install
```

## Compile and run the project

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```
